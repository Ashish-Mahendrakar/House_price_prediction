# 🏠 House Price Prediction using Linear Regression

## 📌 Overview
Built an end-to-end machine learning project to **predict residential property prices** using structured housing data. This project demonstrates strong fundamentals in **data preprocessing, feature engineering, and regression modeling**, with a focus on delivering **interpretable and scalable predictions**.

---

## 🚀 Key Impact
- 📈 Enabled **data-driven pricing decisions** by modeling relationships between housing features and market price  
- ⚡ Reduced manual estimation effort through **automated prediction pipeline**  
- 🔍 Improved model interpretability using **Linear Regression**, making insights actionable for stakeholders  
- 🧠 Demonstrates ability to **translate raw data into business value**

---

## 🧰 Tech Stack
- **Python**
- **Pandas** – data manipulation  
- **Seaborn** – exploratory data analysis  
- **Scikit-learn** – modeling & evaluation  

---

## 📊 Dataset
The dataset contains structured housing data with features such as:
- Zoning classification  
- Lot configuration  
- Building type  
- Exterior materials  
- Sale price (target variable)

---

## ⚙️ Workflow

### 1. Data Preprocessing
- Handled categorical variables using **one-hot encoding**
- Removed irrelevant columns (e.g., ID)
- Cleaned and prepared dataset for modeling

### 2. Feature Engineering
- Converted categorical variables into numerical format
- Selected meaningful predictors impacting house prices

### 3. Model Training
- Applied **Linear Regression**
- Train-test split: **80/20**
- Learned relationships between features and target variable

### 4. Evaluation
- Evaluated performance using **R² score**
- Tested model generalization on unseen data

---

## 📈 Results
- Achieved a desecnt **R² score**, indicating underfiting case
